find_map Find the MAP estimate for a given NumPyro model using SVI with Adam optimizing the ELBO

Args:

rng_key (jax.random.PRNGKey): RNG Key to be passed to SVI.run(). numpyro_model (callable): Python callable containing numpyro.primitives/ Niter (int, optional): Number of iterations to run variational inference. Defaults to 100. lr (float, optional): learning rate used for Adam optimizer. Defaults to 0.01.

Returns:

SVIRunResult.params: parameters of the result of MAP optimization
