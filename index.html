API:

cosmology

a module for basic cosmology calculations using jax.

distributions

a module for basic distribution pdf calculations with jax

interpolation

a module for interpolation calculations using jax

models

a module that stores defined population models

numpyro_distributions

a module defining additional numpyro distributions

pipeline

a module with utilities for running analysis pipelines

postprocess

a module for all postprocessing tools

preprocess

a module for all preprocessing tools
